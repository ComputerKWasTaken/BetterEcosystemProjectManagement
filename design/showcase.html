17 Code & Content

Inline Code

Use the bdWidget() function to create widgets. Pass a type parameter like 'bar' or 'stat'.

Code Block

// BetterScripts widget example
const modifier = (text) => {
  let widgets = '';
  widgets += bdWidget('hp-bar', {
    type: 'bar',
    label: 'HP',
    value: state.game.hp,
    max: 100,
    color: 'var(--bd-success)',
    align: 'left'
  });
  return { text: text + widgets };
};
