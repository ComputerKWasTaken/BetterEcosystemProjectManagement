# Host Settings

> The host of a multiplayer game controls AI settings and permissions for all players.

## Overview

In multiplayer, the host's configuration applies to everyone. Understanding this helps hosts set up appropriate games and helps guests know what to expect.

## What the Host Controls

### AI Model

The host's selected AI model is used:
- All generations use host's model
- If host has premium model access, all benefit
- Guests don't need premium to use host's premium model

### Context Length

Host's context length setting applies:
- Larger context = more story remembered
- Based on host's subscription tier
- Guests benefit from host's tier

### Advanced Settings

All generation parameters:
- Temperature
- Top-P/Top-K
- Response length
- Other model settings

### Content Safety

Safety level from host:
- Safe/Moderate/Mature
- Guests operate under host's setting
- Discuss preferences beforehand

## Premium Benefits Sharing

Key feature: Premium hosts share benefits:

| Host Tier | Guests Get |
|-----------|------------|
| Free | Free features only |
| Adventurer | Adventurer features |
| Champion | Champion features |
| Legend | Legend features |
| Mythic | Mythic features |

This means a premium host can invite free players who then get access to premium models and features during that session.

## Player Management

### Kicking Players

Removes a player temporarily:
- They leave the game
- Can rejoin if they have the code
- Use for disruptions you want to pause

### Blocking Players

Removes a player permanently:
- They cannot rejoin this game
- More serious than kick
- Use for players you don't want back

### Viewing Players

Game menu shows all current players:
- Avatars and names
- Access management options via "..."
- See who's in the session

## Game Settings

### Third Person Mode

Recommended for multiplayer:
- Distinguishes player characters
- Each player has a name
- More readable AI output

### Scenario Selection

Any scenario works, but consider:
- Multiple character roles available
- Third Person enabled
- Appropriate content level

## During Play

### Settings Changes

Host can adjust settings mid-game:
- Model changes apply to next generation
- Parameter changes take effect immediately
- Guests see whatever host configures

### Join Code Persistence

The code remains valid:
- Share anytime during the session
- New players can join ongoing games
- Code works until game ends

## Communication

AI Dungeon doesn't have built-in voice/text chat:
- Use external communication (Discord, etc.)
- Coordinate turns verbally
- Discuss story direction outside the game

## Ending the Game

When host exits:
- The game persists
- Host can return via Continue
- Game technically remains joinable

To fully end access:
- Don't share the code
- Kick/block unwanted players
